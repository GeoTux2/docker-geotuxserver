QGIS Server

QGIS Server

QGIS Server ofrece un servicio web de mapas (WMS y WFS) basado en la librerías de la aplicación de escritorio de QGIS. Se pueden publicar mapas y plantillas de impresión creadas en QGIS como mapas web simplemente copiando el fichero de proyecto QGIS en el directorio del servidorr, y ver así los mapas web exactamente como en el escritorio. QGIS Server normalmente es ejecutado como un módulo CGI/FastCGI dentro de Apache Webserver.

Mapserver

Mapserver

MapServer es un motor de renderización de datos geográficos Open Source escrito en C. Más allá de la navegación por datos GIS, MapServer permite crear “mapas de imágenes geográficas”, es decir, mapas de contenidos para los usuarios.

MapCache

MapCache

MapCache es un servidor que implementa el caché de teselas para acelerar el acceso a las capas WMS. Los objetivos principales es ser rápido y fácil de implementar, esperada de una solución de almacenamiento en caché de teselas.

Tile Server PHP

Tile Server PHP

Alojamiento de carpetas MBTiles y MapTiler con TileJSON, OGC WMTS, interacción UTFGrid e interfaz web. Compatible con QGIS y ESRI ArcGIS. Se ejecuta en cualquier alojamiento web Apache+PHP. Alojamiento de MapBox Studio Vector Tiles.

pycsw

pycsw

pycsw una implementación en python del Estándar de servicio de catálogos web (CSW) de OGC. El proyecto está certificado como que cumple con los estándares OGC, y es una referencia de implementación OGC. El estándar CSW define interfaces comunes para encontrar, explorar y consultar metadatos sobre datos espaciales, servicios web y recursos relacionados.

Lizmap Web Client

Lizmap Web Client

La aplicación de Lizmap Web genera dinámicamente una aplicación visor de mapas web (php/html/css/js) con la ayuda de QGIS Server (QGIS como Servidor OGC). Puede configurar un mapa web por proyecto QGIS con el plugin LizMap de QGIS. La aplicación web Lizmap debe estar instalada en el servidor.

TerriaJS Map

TerriaJS Map

TerriaJS es un Framework de código abierto para construir exploradores de datos espaciales federados para la web. Reúna su catálogo de conjuntos de datos geoespaciales con una combinación de despliegue y consulta de servicios de catálogos en línea, como CKAN, CSW, WMS, WFS y ArcGIS.
